A modern, minimal portfolio starter built with Vite + React. Use this project to showcase projects, skills, and contact information.

## Features

- Sections for About, Projects, Skills, Contact
- Fast development server via Vite

## Tech Stack

- CSS (replace with Tailwind/SCSS as desired)

## Quick start
Requirements: Node.js v16+ and npm.

Clone and run locally:

```powershell
git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>
npm install
npm run dev
```

- Dev server: open the URL printed in the terminal (usually `http://localhost:5173`).
- Build production bundle: `npm run build`.
- Preview production build: `npm run preview`.

Note: This repository is public — avoid committing secrets or personal data. Replace `<your-username>` and `<your-repo>` with your actual GitHub values.

## NPM scripts
- `dev` — Run Vite dev server
- `build` — Build for production
- `preview` — Preview the production build locally

## Project structure
- `index.html` — Vite entry HTML
- `package.json` — Scripts and dependencies
- `vite.config.js` — Vite configuration (React plugin)
- `src/` — React source files
	- `main.jsx` — App bootstrapping
	- `App.jsx` — Main app component
	- `components/` — Reusable UI components
- `public/` — Static assets

Adjust structure to suit your needs (TypeScript, routing, styling, etc.).

## Deployment
This app builds into a static bundle that can be deployed to GitHub Pages, Netlify, Vercel, or any static host.
