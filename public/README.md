# Public Assets Folder

Place your profile picture here with the filename: `profile-pic.jpg`

The image should be:
- Square or close to square aspect ratio (recommended: 512x512px or larger)
- Professional headshot
- JPG, PNG, or WebP format
- Optimized for web (under 500KB recommended)

The profile picture will be displayed in the Hero section with a circular frame and sci-fi styling effects.
